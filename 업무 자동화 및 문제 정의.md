 PASS 조건 : 근거가 충분하고 사실 검증이 완료된 경우
사실과 추정이 명확히 구분되어 있으며, 검증 가능한 내용이 정확하게 서술된 경우

 Fail 조건 : 출처가 없는 경우
사실과 다른 주장이 포함된 경우
추정 내용을 사실처럼 단정한 경우
검증 불가능한 내용을 확정적으로 서술한 경우
 
 

판정 절차 : 제시된 근거와 출처를 검토한다.
검토 결과에 따라 Pass 또는 Fail로 판정한다.
판단이 애매한 경우 담당자가 추가로 재검토한다.
